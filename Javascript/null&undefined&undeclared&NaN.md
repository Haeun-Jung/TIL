## null
- **빈 값**
- null이라는 빈 값을 할당했을 때 생기는 타입이다.

<br/>

## undefined
- **정의되지 않음**
- var 선언문의 경우, 호이스팅 되었을 때 변수 선언과 초기화가 동시에 일어나기 때문에 변수가 undefined 된다.(타입 결정 X)

<br/>

## undeclared
- **선언되지 않음**
- let, const 선언문의 경우, 호이스팅 되었을 때 변수 선언과 초기화가 따로 이루어지기에 변수가 undeclared 되어 에러가 생긴다.

<br/>

- **표현 불가능한 수치형 결과**
