# 스크립트 언어

    ✔ 코드를 작성함과 동시에 인터프리터가 기계어로 번역하고 실행한다.
    ✔ 실행 과정 상에 번역 과정이 있어 비교적 느리다.
    ✔ 문법적 제약이 일반적으로 적다.
    ✔ 운영체제를 신경 쓸 필요 없이 한 번만 작성하면 된다.

- Python
- Javascript
- PHP
- Ruby

<br>

# 컴파일 언어

    ✔ 실행 전 소스코드를 컴파일하여 기계어로 변환 후 해당 파일을 실행한다.
    ✔ 이미 기계어로 변환된 것을 실행하므로 비교적 빠르다.
    ✔ 문법적 제약이 일반적으로 많다.
    ✔ 운영체제에 따라 다르게 작업해야 한다.

- C
- C++
- Java
